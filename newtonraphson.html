Explorable Explanation: The Newton-Raphson Method

Creator: Raycelle Garcia

An interactive Explorable Explanation (EE) to teach about the Newton-Raphson Method as a way that computers solve equation. This live prototype EE was developed for the course "Designing Explorable Explanations for Learning".

I was solely responsibilities for the creation of this EE, which included:

  • Brainstorming and researching topics and solutions
  • Researching and integrating learning mechanics and UX design.
  • Designing the EE content and flow
  • Coding the EE using D3.js, a javascript library for producing interactive data visualizations in web browsers.
